The category of finite sets🔗

Throughout this page, let nnn be a natural number and [n][n][n] denote the standard nnn-element ordinal. The category of finite sets, FinSets\finsetsFinSets, is the category with set of objects N\bb{N}N the natural numbers, with set of maps Hom(j,k)\hom(j,k)Hom(j,k) the set of functions [j]→[k][j] \to [k][j]→[k]. This category is not univalent, but it is weakly equivalent to the full subcategory of Sets\setsSets on those objects which are merely isomorphic to a finite ordinal. The reason for this “skeletal” definition is that FinSets\finsetsFinSets is a small category, so that presheaves on FinSets\finsetsFinSets are a Grothendieck topos.
