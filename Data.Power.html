Power Sets🔗

The power set of a type XXX is the collection of all maps from XXX into the universe of propositional types. Since the universe of all nnn-types is a (n+1)(n+1)(n+1)-type (by n-Type-is-hlevel), and function types have the same h-level as their codomain (by fun-is-hlevel), the power set of a type XXX is always a set. We denote the power set of XXX by P(X)\bb{P}(X)P(X).

ℙ : Type ℓ → Type (lsuc ℓ)
ℙ X = X → n-Type _ 1

ℙ-is-set : is-set (ℙ X)
ℙ-is-set = hlevel 2

The membership relation is defined by applying the predicate and projecting the underlying type of the proposition: We say that xxx is an element of PPP if P(x)P(x)P(x) is inhabited.

_∈_ : X → ℙ X → Type _
x ∈ P = ∣ P x ∣

The subset relation is defined as is done traditionally: If x∈Xx \in Xx∈X implies x∈Yx \in Yx∈Y, for X,Y:P(T)X, Y : \bb{P}(T)X,Y:P(T), then X⊆YX \subseteq YX⊆Y.

_⊆_ : ℙ X → ℙ X → Type _
X ⊆ Y = ∀ x → x ∈ X → x ∈ Y

By function and propositional extensionality, two subsets of XXX are equal when they contain the same elements, i.e., they assign identical propositions to each inhabitant of XXX.

ℙ-ext : {A B : ℙ X}
      → A ⊆ B → B ⊆ A → A ≡ B
ℙ-ext {A = A} {B = B} A⊆B B⊂A = funext λ x →
  n-ua {n = 1} (prop-ext (A x .is-tr) (B x .is-tr) (A⊆B x) (B⊂A x))

Lattice Structure🔗

The type P(X)\bb{P}(X)P(X) has a lattice structure, with the order given by subset inclusion. We call the meets intersections and the joins unions.

maximal : ℙ X
maximal _ = Lift _ ⊤ , λ x y i → lift tt

minimal : ℙ X
minimal _ = Lift _ ⊥ , λ x → absurd (Lift.lower x)

_∩_ : ℙ X → ℙ X → ℙ X
(A ∩ B) x = (∣ A x ∣ × ∣ B x ∣) , ×-is-hlevel 1 (A x .is-tr) (B x .is-tr)
