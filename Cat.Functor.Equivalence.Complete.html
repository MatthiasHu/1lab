Completeness respects equivalences🔗

Let C\ca{C}C be a category admitting limits for J\ca{J}J-shaped diagrams, and F:C≅DF : \ca{C} \cong \ca{D}F:C≅D an equivalence. Then D\ca{D}D also admits limits for J\ca{J}J-shaped diagrams; In particular, if C\ca{C}C is complete, then so is D\ca{D}D.

module
  _ {o ℓ o′ ℓ′} {C : Precategory o ℓ} {D : Precategory o′ ℓ′}
    {F : Functor C D} (eqv : is-equivalence F)
  where

  open is-equivalence eqv

This is a very short theorem: If C\ca{C}C admits J\ca{J}J-shaped limits, then for any diagram K:J→DK : \ca{J} \to \ca{D}K:J→D, the composite F−1KF^{-1}KF−1K has a limit. But since equivalences are right adjoints, FFF preserves this limit, so FF−1KFF^{-1}KFF−1K has a limit in D\ca{D}D; But that composite is naturally isomorphic to KKK, so KKK also has a limit.

  equivalence→complete : ∀ {co cℓ} → is-complete co cℓ C → is-complete co cℓ D
  equivalence→complete ccomp K =
    Limit-ap-iso (F∘-iso-id-l F∘F⁻¹≅Id)
      (subst Limit F∘-assoc (right-adjoint-limit F⁻¹⊣F (ccomp (F⁻¹ F∘ K))))
